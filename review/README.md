# review/（不具合報告用フォルダ）

ここは、実際に書き出したExcelファイルで見つかった不具合（表示崩れ・位置ずれなど）を
確認してもらうために一時的に置く場所です。アプリの動作には使われません
（`templates/schedule_template.xlsx`だけが実際に使われます）。

## 使い方

1. 不具合を再現するExcelファイル（アプリからエクスポートしたもの）をこのフォルダに置く
2. 「reviewに置いたので確認して」のように伝える
3. 中身を調査して原因を特定し、`js/excelExport.js`側のロジックを直接修正する

**このフォルダの中身はあくまで調査用の参考データであり、内容を`templates/`など
他のディレクトリへコピーしたり反映したりすることはしない。**

このフォルダの中身は都度入れ替わる想定なので、確認が終わったファイルは
削除してもらって構いません。
